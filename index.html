Get Started with Git (Local)

git init
Locate and initialize a repository for Git version control
git status
Track changes to files inside the repository
git add <file>
Stage file changes before committing them to repository history
git commit -m "message"
Save a snapshot of staged changes to local repository
git branch <branchname>
Create a new branch that is derived from the current branch
git checkout <branchname>
Switch between branches

Get CPU temperature


cls
$dll = "LibreHardwareMonitorLib.dll"

Unblock-File -LiteralPath $dll
Add-Type -LiteralPath $dll
$monitor = [LibreHardwareMonitor.Hardware.Computer]::new()
$monitor.IsCPUEnabled = $true
$monitor.IsStorageEnabled = $true

$monitor.Open()
foreach ($sensor in $monitor.Hardware.Sensors) {
    Write-Host $sensor.Name
    Write-Host $sensor.SensorType
    Write-Host $sensor.Value
}
$monitor.Close()
